# Parking Management System

A web-based Parking Management System developed using HTML, CSS, and JavaScript.

## Features

- Vehicle Entry
- Vehicle Exit
- Automatic Ticket Generation
- Parking Slot Allocation
- Waiting Queue
- Parking History
- Revenue Calculation
- Live Dashboard

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Data Structures Used

- Arrays
- Objects (Hash Maps)
- Queue
- Stack
